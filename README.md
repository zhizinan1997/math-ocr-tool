# 数学公式图片转LaTeX工具

一个基于AI的数学公式识别工具，支持将截图或图片中的数学公式转换为LaTeX代码，可直接在Word中使用。

## 功能特点

- 🖼️ 支持多种图片格式上传
- 📋 支持粘贴截图功能
- 🤖 支持OpenAI GPT-4V和Claude 3两种AI服务
- 📝 自动生成可用于Word的LaTeX代码
- 👀 实时LaTeX预览
- 📥 一键复制和下载结果

## 快速开始

### 1. 环境要求
- Python 3.8+
- 现代浏览器
